<h2 className="text-xl font-bold text-slate-200">Gotchas</h2>

<ul className="list-disc px-4">
  <li>Dynamically typed</li>
  ```js
  var x = 1;
  x = "hello"
  ```
  <ul className="list-disc list-inside">
    <li>Solution: Use Typescript</li>
    <ul className="list-inside px-8">
      <li>
        ```js
        var x: number = 1; 
        x = "hello; ---> type warning
        ```
      </li>
    </ul>
  </ul>
</ul>


<ul className="list-disc px-4">
  <li>Unchecked nulls</li>
  ```js
  const x = null;
  x.prop
  ```
  <ul className="list-disc list-inside">
    <li>Solution: Use optional chaining and error handling</li>
    <ul className="list-inside px-8">
      <li>
       ```js
    const x = null;
    if (x?.prop); --> handle missing value
    ```
      </li>
    </ul>
    
  </ul>
</ul>
